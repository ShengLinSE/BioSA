Summary

This overview will give you some idea of what data is available, how to find what you need, and what you will see when you make a request to us.

The easiest way to see what data is included is to explore the data. You can build complex queries quickly and browse the results.

Please ensure that you read the guide for polling applications below before starting.

Results are returned in JSON format.

Endpoints

We provide several endpoints to retrieve different items:

  • All Collections
  • One Collection
  • Culture Types
  • Specific Culture

For each endpoint:

  • Results are not filterable at this moment
  • Response contains minimal detail by default but more data can be exposed using parameters in future implementations

Available Parameters case-sensitive

Below are the exposed keys.

  • db
  • collection
  • collections
  • culture
  • cultureType

Parameters Explained case-sensitive

db

Database to query

collection

Returns all documents from a collection

colletions

Returns all documents from all collections

culture

Specifying whether to return monocultures(M) or cocultures(C)

cultureType

Specifying what type of culture type to return, i.e: HA3

Get One Collection

Description Get everything in a collection
Endpoint /query?db={dbName}&collection={collectionName}
HTTP Method GET
Example: /query?db=BIOSA&collection=FEB_23_2018

Get All Collections

Description Get all collections from a database
Endpoint /query?db={dbName}&collections
HTTP Method GET
Example: /query?db=BIOSA&collections

Get Culture Type

Description Get documents for a specified culture type. Use M to get monocultures. Use C get get cocultures.
Endpoint /query?db={dbName}&collection={collectionName}&cultureType={cultureType}
HTTP Method GET
Example: /query?db=BIOSA&collection=FEB_23_2018&cultureType=M

Get Specific Culture

Description Get all documents for a specific culture
Endpoint /query?db={dbName}&collection={collectionName}&culture={cultureName}
HTTP Method GET
Example: /query?db=BIOSA&collection=FEB_23_2018&culture=HA3
